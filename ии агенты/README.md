# 🏝️ Island Survival Agent Simulation

Симуляция выживания AI-агентов на необитаемом острове. Агенты обладают памятью, эмоциями, строят отношения, выживают и общаются друг с другом.


## 🚀 Как запустить проект (на другом компьютере)

### 1. Требования (Prerequisites)
Перед началом убедитесь, что установлены:
- **Node.js** (версия 18 или выше) — [Скачать](https://nodejs.org/)
- **npm** (обычно идет вместе с Node.js)

### 2. Установка (Installation)

1. Клонируйте репозиторий или распакуйте архив с проектом.
2. Откройте терминал в папке проекта.
3. Установите зависимости командой:
   ```bash
   npm install
   ```

### 3. Настройка (Configuration)

Создайте файл `.env` в корневой папке проекта (можно скопировать из `.env.example`).


**Пример `.env`:**
```ini

# Настройки модели
OPENROUTER_BASE_URL=https://openrouter.ai/api/v1
OPENROUTER_DEFAULT_MODEL=arcee-ai/trinity-large-preview:free

# Настройки сервера
SERVER_PORT=8787
VITE_API_BASE=http://localhost:8787
TICK_MS=1500
```

### 4. Запуск (Running)

Запустите проект одной командой (стартует и сервер, и клиент):
```bash
npm run dev
```

После запуска откройте в браузере:
👉 **http://localhost:5173**

## 🎮 Как пользоваться

- **Карта (Canvas):**
    - **ЛКМ**: Выбрать агента или локацию.
    - **Колесико**: Зум (приближение/отдаление).
    - **ПКМ + Драг**: Перемещение по карте.
- **Инспектор (справа):**
    - Показывает состояние выбранного агента (здоровье, голод, инвентарь, мысли).
    - Можно отправить "Приказ свыше" (цель) или вызвать испытание для локации.
- **Панель управления (слева внизу):**
    - Пауза / Старт симуляции.
    - Лента событий (чат, действия).

## ⭐ Основные фишки

1.  **🧠 Умные агенты**: Используют LLM для принятия решений на основе памяти и контекста.
2.  **🗣️ Живое общение**: Агенты обсуждают события, планируют, сплетничают.
3.  **💪 Прокачка навыков**: 5 навыков (Сбор, Крафт, Бой, Медицина, Стройка) с уровнями.
4.  **❤️ Отношения**: Дружба и вражда, визуализированные линиями на карте (зеленые/красные).
5.  **⛈️ Погода**: Динамическая смена погоды (дождь, шторм, туман) с влиянием на геймплей.
6.  **💀 Смертность**: Агенты могут умереть от голода, ран или шторма.
