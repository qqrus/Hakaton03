## 🚀 Быстрый запуск (One-command Deployment)

Проект полностью контейнеризирован. Для запуска всех компонентов (Frontend, Backend, Database) выполните:

```bash
docker compose -f infra/docker-compose.yml up --build -d
```

*   **Frontend**: [http://localhost:3000](http://localhost:3000)
*   **Backend API**: [http://localhost:8000/docs](http://localhost:8000/docs)

## 🛠 Технологический Стек

### **Frontend**
*   **Framework**: Next.js 14+ (App Router, TypeScript)
*   **Design**: **Neo-Brutalism UI** (Vibrant colors, high contrast, bold shadows)
*   **Animations**: Framer Motion
*   **Charts**: Recharts (Динамика баллов и радар компетенций)
*   **Icons**: Lucide-React

### **Backend**
*   **Core**: FastAPI (Python 3.10+)
*   **Database**: PostgreSQL + SQLAlchemy (ORM)
*   **Auth**: JWT (OAuth2 Password Flow)
*   **AI Service**: Integration with **OpenRouter (arcee-ai/trinity-mini)** with native reasoning support.

## 💎 WOW-Фишки и Функционал

### 1. **AI-Аналитика Профиля** 🤖
На странице профиля доступна кнопка "Сформировать аналитику". Нейросеть анализирует историю участий и выдает экспертное резюме на **русском языке**, подсвечивая сильные стороны кандидата.

### 2. **QR-Пропуска** 📱
Автоматическая генерация QR-кодов для каждого мероприятия. Организатор может проверить код, а участник — использовать его как цифровой бейдж.

### 3. **Система Геймификации** 🏆
*   **Уровни**: Novice -> Activist -> Leader -> Elite.
*   **Динамический рейтинг**: Общий лидерборд обновляется в реальном времени.
*   **Матрица компетенций**: Визуализация навыков (IT, Медиа, Лидерство и др.) в виде Radar Chart.

### 4. **Anti-Farming (Защита от накрутки)** 🛡
Внедрена система ограничений: пользователь не может подать более 3 активных заявок одновременно, что предотвращает спам и накрутку опыта.

### 5. **Neo-Brutalist UX** 🎨
Интерфейс не похож на скучные корпоративные системы. Яркий, смелый и современный дизайн привлекает молодёжную аудиторию.

## 🎭 Возможности по Ролям

### **🌟 Участник**
- Просмотр доступных событий и подача заявок.
- Отслеживание статуса своих участий ("Ждем" / "Герой").
- Редактирование профиля (Имя, город, опыт, соцсети).
- Просмотр личной статистики и достижений.

### **🏢 Организатор**
- Управление мероприятиями (Создание, Редактирование, Удаление).
- Просмотр списка кандидатов.
- **Подтверждение участия**: Один клик и участнику начисляются баллы и растет уровень.

### **🧐 Инспектор (HR / Management)**
- Глобальная аналитика по всем пользователям.
- Фильтрация кандидатов по компетенциям.
- **Генерация PDF-отчетов**: Экспорт профиля в PDF с сохранением кириллицы для печати или отправки руководству.
